What is a TinyNAS?

TinyNAS is a tiny web server powered by a ESP32 Wrover module. The web content is serve over the SD card connected to the MCU via SPI link, providing basic HTTP web server function with less than 2W of power usage.

Why TinyNAS?

If you are just hosting a web server for personal use with a few occasional guests, you do not need a Raspberry Pi that draws 15W 24/7 just for this purpose.
Instead, a TinyNAS with ESP32 is just enough to do the tasks. With the help of CDN and Caching, even a 2W MCU running over WiFi can provide relatively acceptable performance for basic HTTP web services.
